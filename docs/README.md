# Screenshots Folder

This folder contains screenshots of the InsureDesk application.

## Required Screenshots

Please add the following screenshots to this folder:

1. **dashboard.png** - Main dashboard with KPIs and statistics
2. **clients.png** - Clients list page showing client management
3. **policies.png** - Policies list page with search and filters
4. **documents.png** - Documents page showing uploaded files
5. **add-policy.png** - Add policy form with autocomplete client search
6. **profile.png** - User profile page

## How to Take Screenshots

1. Run the application: `npm run dev`
2. Navigate to each page
3. Take screenshots (Windows: Win + Shift + S, Mac: Cmd + Shift + 4)
4. Save them in this folder with the names listed above

## Screenshot Guidelines

- **Resolution**: 1920x1080 or higher
- **Format**: PNG (preferred) or JPG
- **Content**: Ensure no sensitive/real data is visible
- **Browser**: Use Chrome/Edge with DevTools closed for clean screenshots
